Privacy Policy

Last Updated: April 19, 2026

1. General Information

Protecting your data is important to us. This privacy policy explains what data Akhi Bot collects, how it is used, and what rights you have.

2. Responsible Entity

The developer is responsible for data processing:

  • Discord: @auserwaehlte
  • Website: akhi-bot-website.vercel.app

3. Collected Data

Akhi Bot stores the following data:

  • Server IDs: For server-specific settings (prefix, language, log channels)
  • User IDs: For leveling, economy, warnings, and moderation
  • Message Content: Only temporarily for the snipe function (max. 1 hour)
  • Voice Activity: For statistics (duration, channel)
  • Role IDs: For auto-role, level roles, team hierarchy

4. Purpose of Data Storage

Data is used exclusively for bot functionality:

  • Leveling system & leaderboards
  • Moderation logs & warnings
  • Welcome messages & auto-roles
  • Custom server settings
  • Statistics & analytics

5. Storage Duration

  • Leveling/Economy: Permanent (until server removes bot)
  • Messages (Snipe): Maximum 1 hour
  • Logs: 30 days
  • Warnings: Until manual deletion

6. Data Sharing

None! Data is never shared with third parties, sold, or used for advertising.

7. Hosting & Database

  • Bot Hosting: Render.com (USA)
  • Database: Supabase (EU Region)

Both providers are GDPR compliant.

8. Your Rights

You have the right to:

  • Access: What data is stored about you?
  • Deletion: Delete your data
  • Correction: Correct incorrect data

Contact the developer via Discord for requests.

9. Data Deletion

Server administrators can delete all server data with !settings reset.
Individual users can request data deletion via DM to @auserwaehlte.

10. Changes

This privacy policy may be updated. The current version is always available at this URL.

11. Contact

For privacy questions:

  • Discord: @auserwaehlte
  • Support Server: discord.gg/yourserver
